Checklist for Thinking Aloud Test

  1. Preparation:
    1. Reset interface for new user.
    2. Check that everything is ready in test room.
    3. Establish protocol for any observers.
  2. Opening:
    1. Greet the participant.
    2. Go through orientation script and set the stage.
    3. Background questionnaire: facilitator asks and fills out form.
    4. Ask user to read and sign consent and non-disclosure forms.
  3. Test Session:
    1. Move over to testing area (computer).
    2. Start session recording.
    3. Provide any prior training.
    4. Provide training of thinking aloud.
    5. User begins with tasks.
    6. User finishes last task.
  4. Closing:
    1. Interview: how was it?
    2. Structured interview questions.
    3. Individual interview questions arising from test.
    4. Feedback questionnaire: user fills out form.
    5. Thank participant, provide any remuneration, show participant out.
  5. Wrap-Up:
    1. Summarise thoughts about this test.
    2. Organise data sheets and notes.
    3. Check and archive session recordings.
