2 Evaluation Methodology

Heuristic evaluation is a process which experts utilize to determine the usability of a user interface by conducting independent walkthroughs and by reporting issues that came up during the walkthroughs. The insights gathered by the evaluators are used to enhance the usability of the user interface of a product, e.g., a website. In 1990 two pioneers in web usability, Jakob Nielsen and Rolf Milch, published an article containing a set of heuristics. This set contains ten heuristics which a system should contain. Firstly the user should be informed appropriately and promptly about the status. Secondly the system should display information in a way that the user is able to understand it. Additionally, it should offer control and make it able for the user to undo errors. The system should also be consistent so the users are not confused and can get familiar with the UI. Errors should also be prevented and if the user wants to engage in risky behaviour, the user should warned beforehand. Information and instructions should be visible so the users do not have to rely on their memory. The UI should also be flexible so experienced users are able to achieve their goals faster. Furthermore, only the most relevant information should be visible to prevent cluttering, which might overwhelm the user. Next, if a error occurs or the user needs help the language should be simple so that the user finds a solution as fast as possible. Finally, the documentation should be easy to find and should also include concise steps for overcoming problems. Even though heuristic evaluation is an extremely powerful tool to enhance the usability of a system, it is so substitute for usability testing. At the end the evaluation is subjective, and the evaluators are influenced by biases which might distort the results.
[IDF]

For this evaluation, the "Andrews General Usability Heuristics", shown in Appendix A.1, will be used. These are based on and slightly adapted from Nielsen's revised set of ten usability heuristics [Nie1994a].

3 User Profiles

Because this website includes an online pharmacy store and also shares information about the pharmacy, the biggest target demographic will be tech savvy people which prefer to buy their medications and other health products online. Additionally, there might a small group of people which can not get out of the house and have to order the things online. If the person is not particularly tech savvy, they might encounter problems. 1) they want to life a more balanced and healty life, they will be reading tips on products to do 2) they want to get the medicine that there doctors gave them, they will more likely only get the medicine and go 3) this group could have some physical obstacle to go get medicine, therefor just order it online 4) they are seeking to be close to nature, therefor read tips on how to improve life with plant based "medicine" and purchase them

4 Extent of the Evaluation

For this evaluation the entirety of the website will be evaluated.
